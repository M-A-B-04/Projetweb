# MyManager - Backoffice de Gestion

## Description

"MyManager" est une application de gestion backoffice construite avec **HTML5**, **CSS3**, et **JavaScript natif** (VanillaJS). L'application permet de gérer plusieurs entités avec des fonctionnalités CRUD (Créer, Lire, Mettre à jour, Supprimer) et propose un tableau de bord interactif et moderne.

---

## Fonctionnalités principales

### CRUD (Clients, Produits, Commandes, Factures, Utilisateurs)
- **Créer** : Ajout de nouvelles entrées via un formulaire dynamique.
- **Lire** : Visualisation des données sous forme de tableaux.
- **Mettre à jour** : Modification des données existantes directement via une interface interactive.
- **Supprimer** : Suppression des données avec confirmation utilisateur.

### Tableau de bord
- Statistiques dynamiques sur les entités.
- Navigation simple via une barre de navigation verticale.
- Design moderne et responsive.

### Génération de données
- Utilisation de **Faker.js** pour simuler des données réalistes.

### Navigation
- Barre latérale verticale pour naviguer entre les entités.
- Bouton "Ajouter une nouvelle entrée" accessible directement dans l'interface principale.
